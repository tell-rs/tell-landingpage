Pipeline

Ingest anything. Route anywhere.

High-throughput Rust pipeline with sub-millisecond latency. Route from any source to any sink with fan-out, filtering, and transforms.

Sources

  • TCP Binary, high-throughput
  • Syslog TCP & UDP
  • HTTP JSONL & Binary

Routing & Transforms

  • Fan-out Multiple destinations
  • Filter By source, type, workspace
  • Pattern Log clustering

Sinks

  • ClickHouse
  • Parquet / Arrow
  • Disk Binary, plaintext, LZ4
  • Forwarder

CLI

Tail and query from terminal

Live stream with filters. Query events with SQL. Pipe to jq, grep, or your LLM.

  • Filter by workspace, source, type
  • Sampling and rate limiting
  • SQL queries via ClickHouse or Polars

# Live tail with filters

$ tell tail --source tcp --type event

# Query events

$ tell query "SELECT event_name, COUNT(*) FROM events GROUP BY 1"

# Pipe to AI

$ tell tail --limit 100 | claude "summarize errors"
